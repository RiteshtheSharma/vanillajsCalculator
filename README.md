# Calculator 
![Screenshot](https://raw.githubusercontent.com/RiteshtheSharma/vanillajsCalculator/main/Screenshot%20from%202024-03-12%2018-47-41.png)

Web calculator for [The Odin Project](https://www.theodinproject.com/).


[Live Website](https://riteshthesharma.github.io/vanillajsCalculator/)  

[Source Repository](https://github.com/RiteshtheSharma/vanillajsCalculator) 

live at https://riteshthesharma.github.io/vanillajsCalculator/


## Features

- Perform basic arithmetic operations
- Runs on immediate execution logic
- Toggle between positive and negative numbers
- Backspace available
- Chainable operations
- Truncated results
